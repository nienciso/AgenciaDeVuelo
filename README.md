# AgenciaDeVuelo
Proyecto Final 2Do año Bios - Se desea realizar el desarrollo de un sistema de promoción de paquetes de viajes para una agencia 
de turismo (empresa que gestiona la información a publicar para sus promociones).Se implementan metodologias con MVC para formularios de empleado y ASPX para formularios publicos. Utilizando patrones de diseño Fabrica y Singleton. SQL Server utiliza Stored Procedures para las acciones.
